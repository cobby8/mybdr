# BDR 시안 의뢰 — 팀/선수 기록 (경기별·대회별·시즌별)

> 디자인 클로드(BDR v2.X)용 의뢰서. NBA.com 선수/팀 페이지 참고. **백엔드 실측 완료 — 신규 DB 0(시즌 박스평균은 매치 집계).**

## 1. 컨텍스트
- 대상: ① 팀 페이지 `/(web)/teams/[id]` ② 선수 개인 `/(web)/users/[id]` — **기록 섹션/탭** 신설·확장.
- 우선순위 P1 · 박제 등급 C/D → A.
- 목적: 팀 전체 선수 / 개인 선수의 기록을 **경기별·대회별·시즌(연도)별** 세 단위로 열람(NBA.com Game Log / Season averages / Splits 경험).

## 2. 변경 요구사항
### 2-1. 핵심
- **3단위 세그먼트 토글: [경기별] [대회별] [시즌별]** — 같은 테이블 골격에 집계 단위만 전환.
  - 경기별 = 매치 단위 박스스코어 로그(Game Log).
  - 대회별 = 대회 단위 합계/평균.
  - 시즌별 = 연도(season_year) 단위 합계/평균 + 시즌 요약 + 슛차트.
### 2-2. 신규/확장 컴포넌트
- (a) **세그먼트 토글** + 공통 sortable 테이블.
- (b) **개인 페이지 — 시즌별 뷰**: 시즌 요약(경기수·승·패·평점·MVP·순위) + **슛차트(존별 효율)** + 시즌 박스평균.
- (c) **팀 페이지 — 로스터 집계 테이블**: 팀 소속 전체 선수 × 동일 3단위 토글.
- (d) 경기별 행 클릭 → 해당 경기 박스스코어/리포트.
### 2-3. 보존(변경 금지)
- AppNav frozen · **기존 시즌 스탯 카드 / 소속 팀 카드 / 레이더 차트(`user-radar-section`) / `user-stats-section`** 는 보존하고 그 아래/탭으로 확장 · 디자인 토큰.

## 3. 디자인 명세

- 데스크톱: 와이드 테이블(가로 스크롤) · 시즌별 슛차트는 좌(코트 존)·우(테이블) 2열.

## 5. ★ 백엔드 실측 — 데이터 출처 (신규 DB 0)
- **경기별**: `MatchPlayerStat` 행 = 매치 박스스코어(전 컬럼). 그대로 로그.
- **대회별**: `MatchPlayerStat.groupBy(tournament)` 합계/평균(의뢰①과 동일 엔진).
- **시즌(연도)별**:
  - 박스 평균/합계 = `MatchPlayerStat` 를 **season_year(매치 날짜) 기준 집계** — ⚠️ 시즌 박스평균은 사전 저장 X, 매치에서 계산(신규 스키마 0, 집계 로직만).
  - 시즌 요약 = `UserSeasonStat`(season_year·games_played·wins·losses·avg_rating·mvp_count·rank_position) **이미 보유**.
  - 슛차트 = `ShotZoneStat`(season_year·zone_code: RA/PAINT/MID_*…·attempts·made·efficiency) **이미 보유** → NBA식 존별 효율.
- **선수 연결**: `TournamentTeamPlayer.userId` → 개인 집계. **팀** = 글로벌 `Team` 로스터(`TeamMember.userId`) → 각 멤버 `MatchPlayerStat`.
- **기존 자산(재사용/확장)**: `/api/v1/players/[id]/stats` · `/api/web/profile/season-stats` · `/api/v1/matches/[id]/stats` · `users/[id]` 의 `user-stats-section`·`stats-detail-modal`·`user-radar-section`.
- **제약(명시)**: ① 개인 박스 기록은 **클레임 선수(userId 보유)** 한정 — 미클레임은 대회 내 `player_name` 만. ② 시즌 박스평균은 매치 집계 결과. ③ 팀 집계는 글로벌 Team 로스터 기준(대회별 TournamentTeam 과 별개).

## 6. 참고 — NBA.com
- NBA.com 선수 페이지: **Game Log(경기별 로그)** · **Splits/Season(시즌별 평균)** · **Shot Chart(존별)**. 팀 페이지: 로스터 + 팀/선수 스탯 + 게임로그.
- 본 의뢰의 3단위 토글 = NBA의 이 구조를 한 화면에서 전환하는 압축형.

# 현재 디자인
### 수정 필요

1. 현재 컬러는 유지해도 될듯. 
글씨만 하얀색으로 변경해서 가독성 좋게 가면 좋을듯. 
2. 첨부했던 코트사진 반영하고, 코트위에 뛰는 선수들. 하단부에 벤치 선수들 배치 
3. 오른쪽은 기존처럼 로그관리가 되는 영역 유지 
4. 선수 클릭하면 에어 커맨드로 스탯 입력방식 (현재 버튼 방식은 폐기)
5. 선수교체는 드래그로 바꿀수 있게 함. 
6. 오른손으로는 태블릿 펜슬을 들고 에어 커맨드를 사용할거고, 왼손으로는 샷클락 시간 조정등을 할거라. 왼손이 닿는 영역은 현재 하단부 버튼처럼 큼지막하게 만들어서 위치도 신경써줘.

## 2. 핵심 변경 (v1 → v2)

### 2-1. 종료 대회 = **진행 중 대회와 동일 탭 구조 / 첫 탭만 교체**

```
[진행 중 대회 (TournamentDetail.jsx · 운영 carry)]
  탭 1: 대회소개  / 경기일정 / 대진표 / 참가팀 / 규정

[종료 대회 (TournamentCompleted.jsx · v2 박제 대상)]
  탭 1: **대회결과** ← 신규 (대회소개 자리)
  탭 2: 경기일정   ← 진행 중 UX carry (풍부한 카드 + sidebar)
  탭 3: 대진표    ← 진행 중 UX carry
  탭 4: 참가팀    ← 진행 중 UX carry
  탭 5: 규정      ← 진행 중 UX carry
```

→ **종료 ↔ 진행 중 = 첫 탭(대회결과 vs 대회소개) 만 차이 / 나머지 4 탭 완전 동일 UX**.

### 2-2. "대회결과" 탭 콘텐츠 = 현행 결산 5카드 통합

```
[대회결과] 탭 안 (기본 선택 / first)
  ┌─ Hero band: 🏆 우승팀 (chamiponship 모드 — 첨부 5번째 캡처)
  │   - 우승팀 logo + 이름 / 대회 메타 (장소·날짜·종별)
  │   - stat 4 (엔트리 / 경기 / 평균 득점 / 평균 마진)
  │
  ├─ 섹션 01. 최종 순위 (좌 컬럼)
  │   - 우승 / 준우승 / 공동 3위 카드 list
  │   - "전체 N강 보기 →" link
  │
  ├─ 섹션 02. MVP · 베스트5 (우 컬럼)
  │   - MVP Hero (선수 + 팀 + 평균 득점 + 어시)
  │   - 베스트5 list (포지션별 chip + 선수)
  │
  ├─ 섹션 03. 명장면 갤러리 (좌 컬럼)
  │   - 사진 grid 6 (raw 또는 placeholder)
  │   - "전체 보기 →" link
  │
  ├─ 섹션 04. 대회 알기자 (우 컬럼)
  │   - "커뮤니티 연결" badge
  │   - 종료 발표문 카드 (제목 + 본문 + "전문 보기 →" CTA)
  │
  └─ 섹션 05. 다음 회차 (전체 폭)
      - D-N + 시리즈 이름 (Vol.N) + 시작일
      - "알림 받기" CTA
```

→ 첨부 5/6번째 캡처 그대로 carry. 별 레이아웃 변경 ❌.

### 2-3. 다른 탭 (경기일정·대진표·참가팀·규정)

진행 중 대회 (`TournamentDetail.jsx`) 의 동일 컴포넌트 / 동일 UX **그대로 재사용**.

- **경기일정 (4번째 캡처)**: 종별 filter chip / 일자별 그룹 / 경기 카드 (시간 + 팀 + 점수 + status) / 우측 sidebar (참가비 / 접수현황 / 우승 예측)
- **대진표**: 진행 중 carry / 종료 대회는 결과 표시 (✅ 완료 매치)
- **참가팀**: 진행 중 carry / 종료 대회는 최종 결과 badge (우승 / 준우승 / 공동 3위 / 8강 · 첨부 3번째 캡처)
- **규정**: 진행 중 carry / 변경 ❌

→ **종료 대회 특수 표시**: 매치 status badge "종료" / 팀 카드 "최종 결과 badge" / "FULL RECORD" 헤더는 제거 (탭 통합으로 의미 소실).

## 5. Interaction states

- **default**: 첫 탭 = 대회결과 (status='completed' 진입 시 자동 선택)
- **empty**: 결산 데이터 없는 카드 = 카드 hide (mock ❌). 일정/대진표/참가팀 = "기록 없음"
- **loading**: 탭 클라이언트 패칭 스켈레톤
- **종료 표시**: 경기 카드 / 매치 / 팀 = "종료" badge 통일

경기 개설 — 단일 페이지 + 라이브 프리뷰

입력하면서 우측에 실제 카드 미리보기 가 실시간 업데이트되는 단일-페이지 폼입니다. 카드의 다크 컬러 시스템과 종별 hover 인식을 그대로 가져와 "어떤 카드가 만들어지는지" 를 눈으로 보면서 채우는 흐름이 핵심입니다.

핵심 디자인 의사결정
  • 단일-페이지 폼 (반복 호스트는 빠르게, 신규는 sticky 미리보기로 안내)
  • 2-column PC (form 1fr + sticky preview 360px) / 1-column Mobile (상단 collapsible preview)
  • 종별 선택 시 컬러 즉시 반영 — 폼 전체 accent (kind) 컬러 전환
  • 날짜·시작·종료 자동 duration 계산 (예: 12:00–15:00 → 3시간)
  • 코트 선택 — DB 검색 picker (이번 시안은 선택 완료 상태 mock)
  • 정기 모집 토글 → 요일 선택 picker 노출
  • 모바일 하단 fixed bar — 취소 / 임시저장 / 개설

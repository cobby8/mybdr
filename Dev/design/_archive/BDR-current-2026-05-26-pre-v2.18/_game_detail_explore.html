경기 상세 페이지 — 시안 2종

카드에서 확정된 디자인 언어(다크 네이비/크림슨/포레스트 + Pretendard 900 + 주최자 아바타)를 상세 페이지로 확장합니다. 현행 2-column 구조를 정제한 V1 (Classic Refined) 과 풀폭 hero 를 도입한 V2 (Hero-led) 를 비교합니다.

의뢰 응답 반영
  • 코트 사진 ❌ → 지도 미리보기 로 대체 (Kakao/Naver Map embed 자리)
  • 참가자 정보: 닉네임 + 레벨 + 포지션 + 평점 풀-노출
  • 호스트 문의: 댓글 + 쪽지 둘 다 — 인라인 댓글 + 우측 패널 쪽지 버튼
  • 추가 섹션: 같은 코트 다른 경기 / 주최자 다른 모집 / 참가자 후기 / 코트 시설
  • 상태 6종: 기본 / 마감임박 / 마감 / 신청함(대기·승인) / 호스트 본인 / 미로그인
  • 모바일: 하단 fixed sticky bar (정원 + 신청 버튼)

BDR v2.16 · DATE TILE 세부 시안

Date Tile — 한글 / 지역 강조 / 주최자 아바타

이전 시안 B(Date Tile) 를 기반으로 PM 의 3 가지 요구사항을 반영한 세부 시안입니다. 변동 요소는 지역 표시 강도 만 다르게(B1 굵은 텍스트 / B2 컬러 chip / B3 좌측 컬러 막대 전용 라인) — 나머지는 동일합니다.

반영 요구사항
  • 영문 라벨 (APR / SAT) 제거 → 한글 "4월 / 26 / 토" 표기
  • 지역 표시 강화 — 3 가지 방식 비교
  • 좌하단 주최자 아바타 + 닉네임 배치
B1

Area Bold — 굵은 텍스트 + pin 아이콘

가장 가벼움 시각 잡음 ↓

지역을 종별 chip 옆에 본문 색·800 weight 로 표시. 종별 컬러 핀(📍) 으로 시각적 결합. chip 보다 가볍지만 일반 텍스트보다 무게 있음.

정리

3 시안 비교

시안 지역 표시 시각 무게 추천 케이스
B1 800 weight 텍스트 + pin 가벼움 정보 밀도 우선, 모바일 줄바꿈 회피
B2 회색 chip + 종별 컬러 pin 중간 관용적 — 학습비용 ↓, 필터링 메타데이터처럼 보임
B3 전용 라인 + 컬러 막대 강함 지역 기반 탐색이 핵심 일 때 가장 효과적

개인 추천: B3 가 "전국 농구 매칭 플랫폼" 의 핵심 가치 (지역 기반 탐색) 와 가장 잘 맞습니다. 다만 B1 / B2 도 다른 강조 우선순위가 있을 때 적합합니다.

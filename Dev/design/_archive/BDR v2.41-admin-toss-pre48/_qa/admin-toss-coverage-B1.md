## 5. 모바일 최적화 전수 점검 (v2.41 B1)

> 5개 HTML(대회 생성/수정/운영 + 대회 관리자 + 백오피스) 전 화면 점검.

### 5-1. 셸 공통 (admin-shell + toss.css)
- ✅ 사이드바 → ≤900px 토픽바 + 드로어 자동 전환, body 스크롤 락
- ✅ `.ts-main` margin 해제 + 56px 토픽바 패딩, `.ts-main__inner` ≤720px 패딩 축소

### 5-2. 이번 점검에서 보정한 항목
- 🔧 **데이터 테이블 가로 스크롤** — 고정폭 컬럼(84~110px) 다수라 ≤760px에서 fr 컬럼이 0으로 붕괴·`overflow:hidden`에 클립되던 문제. `.ad-tablescroll`(overflow-x:auto) 래퍼 + ≤760px `min-width:660px`로 가독폭 유지 + 가로 스와이프. (admin-shell `DataTable`, admin-pages.css)
- 🔧 **iOS 입력 줌 방지** — `.ad-search input` 14.5 → **16px** (포커스 시 자동 확대 제거)

### 5-3. 기존 정상 확인
- ✅ KPI 그리드 4 → 2(≤900) → 1(≤480)
- ✅ 카드 그리드 3 → 2(≤900) → 1(≤600)
- ✅ 대시보드 2컬럼 → 1컬럼(≤900)
- ✅ 툴바(검색+필터칩) flex-wrap, 검색 min-width:0
- ✅ 워크스페이스(생성/수정/운영) — 단계바/저장바/일정·대진 그리드 전부 ≤720 분기 보유(이전 범위 감사 완료)
- ✅ `.ts-input` 16px(폼 줌 없음) · 버튼 터치 타겟 ≥44px(`.ts-btn` 패딩 14px)

# 관리자 전역 데드버튼·미구현 감사 (B1)

> 4개 관리자 셸(대회 관리자 / 백오피스 / 협력업체 콘솔 / 심판 관리자) + 운영 워크스페이스 전수 점검.
> 운영 워크스페이스(`대회 운영.html`)는 이미 전 버튼 실배선(onClick+토스트) — 손대지 않음.

## 1. 공통 인프라 추가
- **글로벌 토스트** — `AdminShell` 에 `window.adToast(msg)` 배선(`.ts-toast` 하단 중앙, 2.2s 자동 소멸). 모든 셸 공용.
- **DataTable `onRow`** — 행 클릭 네비게이션 지원(클릭 가능 시 cursor pointer).

## 2. 미구현/데드버튼 → 처리 결과

### 대회 관리자 (`ta-pages.jsx`)
| 위치 | 이전 | 처리 |
|---|---|---|
| 대회 목록 행 | "행을 눌러 운영 이동" 안내만, 클릭 무반응 | ✅ 행 클릭 → `대회 운영.html` 실이동 |
| 행 액션 운영/사이트 | 무반응 | ✅ 운영 → `대회 운영.html` · 사이트 → 토스트(공개 사이트 미구현 명시) |
| 새 대회 만들기 | 무반응 | ✅ `대회 생성.html` 이동 |
| 내보내기 | 무반응 | ✅ 토스트(시연) |
| 단체·주최: 단체 등록/관리/운영진 | 무반응 | ✅ 토스트 |
| 시리즈: 만들기 / 행 | 무반응 | ✅ 토스트 / 행 클릭 토스트 |
| 템플릿: 만들기 / 이 템플릿으로 생성 | 무반응 | ✅ 토스트 / `대회 생성.html` 이동 |

### 백오피스 (`bo-pages.jsx`) — 공용 블록으로 리팩터
- 기존 사설 `renderCell`/`SchemaPage`/`Dashboard 막대`/`Analytics`/`Settings` **중복 제거** → `admin-blocks` 공용(SchemaList·AdBarPanel·AdListPanel·AdSettings)으로 통합. 중복 데드버튼 일괄 해소.
- 리포트 내보내기 · 기간 선택 · 설정 저장/취소 → ✅ 토스트.

### 공용 블록 (`admin-blocks.jsx`) — 3개 셸(백오피스/협력업체/심판) 동시 적용
| 위치 | 처리 |
|---|---|
| 툴바 내보내기 / +생성(addLabel) / 필터 | ✅ 토스트 |
| 행 액션 보기(eye) / 더보기 | ✅ 토스트(상세 보기·더보기) |
| 설정 저장 / 취소 | ✅ 토스트 |
| 검색창 | 기존 실시간 필터(정상) |

### 협력업체 콘솔 (`partner-pages.jsx`)
- 월간 리포트 · 캠페인 상세 수정/일시중지 → ✅ 토스트.
- 캠페인 목록 행 → 상세 드릴인(기존 정상).

### 심판 관리자 (`referee-pages.jsx`)
- 주간 리포트 · 캘린더 이전/이번주/다음 · 등급·수당 정책 수정 → ✅ 토스트.

## 3. 의도적 미구현(시연 토스트로 표기)
서버 호출/DB 연동이 필요한 동작은 운영 워크스페이스와 동일하게 **"(시연)" 토스트**로 통일.
실제 화면 이동(대회 생성/수정/운영)만 실 네비게이션 처리.

## 4. 미구축 영역(별도 산출물 필요 — 데드버튼 아님)
- 공개 토너먼트 사이트(`_site/*`) 결과물 — 관리자에서 "사이트" 토스트로 안내만.
- 각 row 상세 모달/드로어 — 현재 토스트 피드백. 필요 시 다음 범위에서 상세 패널 추가 가능.

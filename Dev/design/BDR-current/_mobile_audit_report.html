05권장 후속 액션

  • P0 · messages: 모바일 single-column 흐름 (목록 → 채팅 push). 현재 채팅 본문 접근 불가.
  • P0 · 다중 컬럼 board 테이블(teamManage·billing·scrim·match teams)에 data-label 라벨링 패턴 적용. 컴포넌트화 추천: <ResponsiveTable />.
  • P1 · bracket: 라운드 헤더 sticky-top, 또는 모바일 vertical 모드.
  • P1 · gallery 라이트박스: .page 외부의 모달 셀렉터 추가, 또는 모달 내부에 모바일 stack class.
  • P1 · 가로 스크롤 영역(스코어 테이블·쿼터별·대진표) 우측 페이드 마스크 + 첫 진입시 0.5s 잠깐 미리 스크롤하는 인터랙션 hint.
  • P2 · calendar: 셀 탭 → 일간 모달.
  • P2 · postWrite: 툴바 그룹화/스크롤.
  • P2 · profile: 320px 카드 sidebar → 모바일 hero 섹션 변환.
  • P3 · 테이블 컴포넌트화 후 responsive.css의 path-based grid selector를 60% 정도 줄일 수 있음. 유지보수성 ↑.

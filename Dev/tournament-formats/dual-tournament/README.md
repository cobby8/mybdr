# 듀얼토너먼트 (`dual_tournament`)

> 학습 자료 — 9장 사진 분석 (2026-05-02 v1) + 사용자 명확화 (2026-05-02 v2).
> 동호회최강전 (16팀 / 5/2~5/10) 사례 기반.

## 한 줄 정의 (v2 갱신)

**16팀을 4조×4팀으로 분배 → 각 조에서 듀얼토너먼트 5경기로 조 1·2위 결정 → 8강 단판 토너먼트로 우승팀 결정**.

## 사용자 명확화 (2026-05-02)

- **조별 듀얼토너먼트 = 조별 5경기** (옛 "조별 미니 더블엘리미" 라벨 폐기)
  - G1·G2 (1라) + 승자전 G3·패자전 G4 (2라) + 최종전 (3라)
- **조 1위 = 승자전(G3) 승자**
- **조 2위 = 최종전 승자** (= 승자전 패자 vs 패자전 승자 중 승자)
- **최종전까지가 조별 경기** (조별 = round 1·2·3 = 5매치)
- 4조 × 5경기 = **조별 20경기** + 8강 4 + 4강 2 + 결승 1 = **27경기** (불변)

전형 명칭: "조별 더블엘리미 + 8강 단판 토너먼트" 하이브리드.
("듀얼" = dual = 두 갈래 = 조별 단계의 **승자전 + 패자전 두 갈래** 진행 의미)

## 전체 흐름 (16팀 → 우승팀)

### Stage 1: 조 분배 (시드 기반)
- 4조 × 4팀 = `groupCount: 4, teamsPerGroup: 4`
- 시드 배정 패턴 (BDR 랭킹 기반 균형 분배 추정)

### Stage 2: 조별 1일 진행 (4팀 미니 더블엘리미)
각 조 1일 = **4경기**:

| 라운드 | 경기 | 매칭 | 의미 |
|--------|------|------|------|
| R1 | GAME 1 | A1 vs A2 | 1라운드 매치 1 |
| R1 | GAME 2 | A3 vs A4 | 1라운드 매치 2 |
| R2 | GAME 3 | G1 승자 vs G2 승자 | **승자전** (Winner Bracket) |
| R2 | GAME 4 | G1 패자 vs G2 패자 | **패자전** (Loser Bracket) |

→ **승자전 승자 = 잠정 1위 / 패자전 패자 = 4위 확정**.

### Stage 3: 조별 최종전 (2위 결정전)
다른 날, 1경기:

| 매칭 | 의미 |
|------|------|
| 승자전 패자 vs 패자전 승자 | **2위 결정전** (Bronze Final) |

→ **최종전 승자 = 2위 / 최종전 패자 = 3위**.

**조별 최종 순위**:
- 1위 = 승자전 승자
- 2위 = 최종전 승자
- 3위 = 최종전 패자
- 4위 = 패자전 패자

### Stage 4: 8강 단판 토너먼트 (조별 1·2위 8팀)
크로스 브래킷 (다른 조와 매칭):

| 8강 매치 | 매칭 |
|----------|------|
| 8강 1 | B조 1위 vs A조 2위 |
| 8강 2 | A조 1위 vs B조 2위 |
| 8강 3 | D조 1위 vs C조 2위 |
| 8강 4 | C조 1위 vs D조 2위 |

### Stage 5: 4강 + 결승 — 진짜 크로스 브래킷 (NBA 컨퍼런스 모형)

| 매치 | HOME | AWAY | 의미 |
|------|------|------|------|
| 4강 1 | **8강 1 승자** | **8강 4 승자** | B·A 측 ↔ C·D 측 첫 만남 |
| 4강 2 | **8강 2 승자** | **8강 3 승자** | A·B 측 ↔ D·C 측 첫 만남 |
| 결승 | 4강1 승자 | 4강2 승자 | 단판 |

⚠️ **중요**: 4강 매칭은 `1+2/3+4` (단순 순차) 가 아닌 `1+4/2+3` (크로스). 모든 조 1위가 결승 전까지 분산되도록 설계.

**시드 의미별 4강 라인**:
- 8강 1 (B조 1위 vs A조 2위) = 1번 시드 라인 → **4강 1**
- 8강 4 (C조 1위 vs D조 2위) = 4번 시드 라인 → **4강 1**
- 8강 2 (A조 1위 vs B조 2위) = 2번 시드 라인 → **4강 2**
- 8강 3 (D조 1위 vs C조 2위) = 3번 시드 라인 → **4강 2**

**예시 — 모든 조 1위가 8강 승리한 경우**:
- 4강 1: **B1 vs C1** (B조·C조 1위)
- 4강 2: **A1 vs D1** (A조·D조 1위)
- 결승: (B1 또는 C1) vs (A1 또는 D1)

→ 어느 두 조 1위도 4강 전까지 안 만남. 결승 매치업 가능 조합 = 4가지 (A·B, A·C, B·D, C·D).

## 코드 매핑 (현재 vs 필요)

### 현재 상태 (구현 0)
| 위치 | 처리 |
|------|------|
| `tournament-status.ts:70` | "듀얼토너먼트" 라벨 ✅ |
| `bracket-settings-form.tsx:33` | `dual_tournament` 도 `isSingleElim = true` 로 single 동일 처리 ⚠️ |
| `bracket-generator.ts` | `generateSingleEliminationBracket` 만 — **듀얼 전용 함수 0** ❌ |
| `league-generator.ts` | 풀리그 생성 — 듀얼과 무관 |

### 필요 구현
1. **`generateDualTournament(teams, settings)`** 신규 함수
   - 입력: 16팀 + settings (groupCount=4, teamsPerGroup=4)
   - 출력: 27 매치 (조별 16 + 최종전 4 + 8강 4 + 4강 2 + 결승 1)
   - bracketLevel/bracketPosition/roundName/roundNumber 모두 설정
   - next_match_id 연결 (winner→다음 라운드 / 조별 최종전은 바로 8강 X — 조 1·2위만)

2. **조 배정 시드 알고리즘**
   - 시드 1~16 → 4조 분배 (강팀 분산: snake 패턴 또는 시드 mod 4)
   - 동호회최강전 사진 보면 BDR 랭킹 기반 사용자 수동 배정 패턴

3. **2위 결정전 로직**
   - 조별 4경기 완료 후: 승자전 패자 + 패자전 승자 자동 매칭

4. **8강 크로스 브래킷 (조별 1·2위 매칭)**
   - 8강 1: B1 vs A2 / 8강 2: A1 vs B2 / 8강 3: D1 vs C2 / 8강 4: C1 vs D2

5. **4강 크로스 브래킷 (NBA 컨퍼런스 모형)**
   - 4강 1: 8강 1 승자 vs **8강 4** 승자 (단순 1+2 X)
   - 4강 2: 8강 2 승자 vs **8강 3** 승자
   - 효과: 모든 조 1위가 결승 전까지 분산

5. **settings.bracket schema 갱신**
   - 현재 B 대회 settings: `{ groupCount: 2, advancePerGroup: 2, knockoutSize: 8 }` ⚠️ 정합 안 됨
   - 정합값: `{ groupCount: 4, teamsPerGroup: 4, advancePerGroup: 2, knockoutSize: 8, hasGroupFinal: true }`

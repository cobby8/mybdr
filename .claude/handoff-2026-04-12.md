## 🐛 알려진 문제 / 회피법

### Turbopack "Jest worker exceptions" 에러
- **증상**: `Jest worker encountered N child process exceptions, exceeding retry limit`
- **원인**: Turbopack HMR Worker Pool 상태 꼬임 (많은 파일을 단시간에 수정하면 누적)
- **해결**: 
  1. 포트 기반 dev 서버 재시작: `netstat -ano | findstr :3001` → `taskkill //f //pid <PID>` → `npm run dev`
  2. 안 되면: `.next` 폴더 삭제 후 재시작
- **절대 금지**: `taskkill //f //im node.exe` — 다른 프로젝트 + Claude Code까지 죽음
- 상세: `.claude/knowledge/errors.md` [2026-04-12] 항목

### `.env` 운영 DB URL 주의
- 개발 작업 중 `DATABASE_URL`이 운영 DB를 가리키면 **사용자 데이터 파괴 위험**
